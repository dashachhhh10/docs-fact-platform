# Docs-Fact — платформа сверки документов и фактического исполнения перевозок

Веб-платформа для автоматизированной сверки транспортных документов с фактическими данными о выполнении перевозок. Система сопоставляет рейсы, накладные, GPS-события и фотоподтверждения, выявляет расхождения и классифицирует их по степени критичности. Встроенный ML-модуль на TensorFlow.js оценивает риски нарушений на основе исторических паттернов. Платформа позволяет логистическим операторам сократить время ручной проверки документов и снизить количество неурегулированных претензий.

## Стек технологий

- **HTML5** — разметка и структура
- **CSS3** — стилизация, темёная/светлая темы
- **Vanilla JavaScript** — логика приложения, роутинг, движок сверки
- **Chart.js** — графики и диаграммы на дашборде
- **TensorFlow.js** — ML-классификатор расхождений
- **Lucide Icons** — иконки интерфейса

## Как открыть локально

**Вариант 1 — напрямую:**
Откройте файл `index.html` в браузере (Chrome или Firefox).

**Вариант 2 — через локальный сервер (рекомендуется):**
```bash
# Python 3
python3 -m http.server 8080
# Затем откройте http://localhost:8080
```

## Тестовые данные

В папке `test-data/` находятся четыре CSV-файла для демонстрации работы платформы:

| Файл | Содержимое |
|------|------------|
| `trips.csv` | Рейсы: номер, маршрут, перевозчик, даты, статус |
| `documents.csv` | Транспортные накладные и ТТН |
| `gps_events.csv` | GPS-метки с координатами и временными метками |
| `proofs.csv` | Фотоподтверждения доставки |

Загрузите их через экран **«Загрузка данных»** в интерфейсе платформы.
