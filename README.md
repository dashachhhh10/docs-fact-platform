# Docs-Fact

Платформа сверки документов и фактического исполнения автомобильных перевозок.

## Описание

Платформа реализует механизм автоматизированной сверки данных перевозочных документов с фактическими данными исполнения рейса. Сопоставление выполняется в процессе перевозки, а не в момент документального закрытия. При обнаружении расхождений между плановыми параметрами (точки маршрута, временные окна, состав груза) и фактом (GPS-события, отметки водителя, статусы документов) формируются карточки расхождений с классификацией по типу и критичности.

Приоритизация расхождений реализована через комбинацию rule-based логики и нейросетевого классификатора на TensorFlow.js. Документальное закрытие рейса блокируется при наличии открытых критических расхождений.

## Стек

- HTML5, CSS3, vanilla JavaScript (ES6+)
- Chart.js — графики и диаграммы
- TensorFlow.js — нейросетевая классификация приоритетов
- Lucide Icons — иконки интерфейса
- localStorage — персистенция состояния
- Vercel — статический хостинг

## Тестовые данные

В директории `test-data/` находятся четыре CSV-файла, описывающих 12 перевозок с различными типами расхождений.

| Файл | Содержимое |
|---|---|
| trips.csv | Идентификаторы рейсов, маршруты, перевозчики, плановые даты, статусы |
| documents.csv | Транспортные накладные ТТН и ЭТрН с реквизитами |
| gps_events.csv | GPS-события: координаты, временные метки, идентификаторы геозон |
| proofs.csv | Фотоподтверждения и подписи водителей по этапам |

## Функциональные модули

- Загрузка и парсинг CSV-данных
- Матрица сверки «Документы — Факт» по 8 типам правил
- Автоматическая классификация расхождений по критичности (CRIT / WARN)
- Приоритизация обработки (Срочно / До конца дня / Стандарт / Низкий)
- Нейросетевой классификатор приоритетов: Dense(11) → Dense(16, relu) → Dense(8, relu) → Dense(4, softmax)
- Реестр карточек расхождений с управляемым жизненным циклом
- Журнал истории действий по каждой карточке
- Контроль документального закрытия рейса
- Дашборд с агрегированными показателями
- Тёмная и светлая темы оформления
