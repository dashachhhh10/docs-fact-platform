# Docs-Fact

Платформа сверки документов и фактического исполнения автомобильных перевозок.

## Описание

Платформа реализует механизм автоматизированной сверки данных перевозочных документов с фактическими данными исполнения рейса. Сопоставление выполняется в процессе перевозки, а не в момент документального закрытия. При обнаружении расхождений между плановыми параметрами (точки маршрута, временные окна, состав груза) и фактом (GPS-события, отметки водителя, статусы документов) формируются карточки расхождений с классификацией по типу и критичности.

Приоритизация расхождений реализована через комбинацию rule-based логики и нейросетевого классификатора на TensorFlow.js. Документальное закрытие рейса блокируется при наличии открытых критических расхождений.

## Стек

- HTML5, CSS3, vanilla JavaScript (ES6+)
- Chart.js — графики и диаграммы
- TensorFlow.js — нейросетевая классификация приоритетов
- Lucide Icons — иконки интерфейса
- localStorage — персистенция состояния
- Vercel — статический хостинг
